# Instagram-Clone

Instagram is a social media application where people can interact with each other.

## Introduction
Instagram is developed using Flutter and Firebase. Instagram connects you to people around the world on a single tap.

## Features

1. Login
2. Sign-Up
3. Reset Password
4. Add Post
5. Feeds
6. Timeline
7. Search
8. Profile Screen
9. Activity Screen
10. Inbox
11. Messages
12. Online/Offline Status
13. Comment
14. Like
15. Like Comment
16. Like Animation
17. Post Preview
18. Sign-Out
19. Delete Post
20. Camera/Gallery Upload
21. Follow/Unfollow
